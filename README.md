# 📚 Ink & Ivy Bookstore Analysis

A mini Excel project created to practice Microsoft Excel for data analytics.

## Skills Practiced

- Tables
- Formulas
- SUMIF / COUNTIF / XLOOKUP etc
- Pivot Tables
- Pivot Charts
- Conditional Formatting
- Dashboard Design
- Sorting & Filtering

## Files

- `bookstore_sales.xlsx` – Main Excel workbook
- `dashboard.png` – Final dashboard screenshot

## Project Goal

Analyze bookstore sales data and build a simple interactive dashboard to answer business questions like:

- Which genre generates the most revenue?
- Which payment method is used the most?
- Which books are the best sellers?
- What are the monthly sales trends?

This project was created for learning and practicing Excel.
